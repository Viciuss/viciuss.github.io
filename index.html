O que é a Poképedia

A Poképedia foi um projeto criado durante a aula de Analise e desenvolvimento de sistemas com fins pedagógicos baseado em um conteúdo que eu gosto.



Motivações

A motivação inicial para a criação desse site foi principalmente para ganhar nota, mas também para aprimorar meus conhecimentos sobre estruturas HTML e aprender a usar o CSS.

Previsões futuras

Esse texto possivelmentee ficará datado com o tempo.
Eu pretendo implementar algo util nesse site e usa-lo para estudos sobre bancos de dados SQL e noSQL, aprimorando ainda mais meu conhecimento sobre a área back-end
